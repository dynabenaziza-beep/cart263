Project 2 

Data Loop — Interactive Artwork
How It Works
This is an interactive system where the user creates data through simple actions.
Click on the screen to generate data points
Data points move and connect based on similarity
When enough connections form, a new identity is created
Each identity makes the central sphere grow
After 3 identities, the system transitions to the archive
In the archive, all generated identities are stored and displayed as part of a larger dataset.
Controls


Click → create data
R key → restart


Tools
Built with HTML, CSS, JavaScript
Uses Three.js for the 3D sphere

Aydan’s Project
Live link: https://aydanmcgrail.github.io/cart263/Project1_aydan/
GitHub: https://github.com/aydanmcgrail/cart263
I really liked Aydan’s project. I think it was one of the most creative ones. I liked how he used his drawing skills to build everything instead of using simple shapes. It felt very personal and unique, and you can really see his style in the work. What I found really interesting is how he used real objects as metaphors, like the coins becoming clouds and the fork as a button. It made the project feel more meaningful, like it is saying something about money and consumption.
The hands were also really cool and added a lot to the interaction. The whole project felt like an interactive collage, where different objects come together to create a story. I also liked how everything was placed on the screen because it made the project feel clean and well thought out.
One thing I found interesting in the code is how the hand follows the mouse with a small delay instead of moving instantly. It slowly moves toward the mouse, which makes it feel smoother and more natural. It almost feels like the hand has weight. As a student, I liked this because it’s a simple idea, but it really changes how the interaction feels.
